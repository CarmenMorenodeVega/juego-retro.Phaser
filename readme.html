Recordando el Super Pang

Descripción del juego:

El objetivo del juego es explotar unas bolas que se mueven por la pantalla. Los personajes cuentan con dos tipos de armas para ir rompiendo las bolas: un arpón y un arma de fuego. El cambio se produce al recoger una estrella. Por cada impacto de un disparo en una de estas bolas, estas se dividen en dos bolas más pequeñas. Cuando las bolas adquieren un determinado tamaño, dejan de dividirse y desaparecen.
El objetivo principal del juego es acabar con las bolas sin que estas toquen a los personajes. Si una bola toca un personaje, este perderá una vida. Se pretende implementar el juego como un multijugador con puntuaciones, vidas, potenciadores y modificadores de armas.

Es el acceso a un menú en el que se puede seleccionar si jugar de forma local o crear una partida online. Está implementada con una página HTML, ayudado con estilos de css y bootstrap.

Pantalla: Elegir mundo

En primera instancia se debe elegir el mundo al que se quiere acceder mediante un sprite que se desplazará on mouse. Una vez se elige el mundo donde se quiere jugar, se cargan las imagenes que aparecerán en el anvas como background y se permite elegir el modo de juego.
